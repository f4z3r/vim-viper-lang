INTRODUCTION                  *viper-introduction* *vim-viper-lang-introduction*

The vim-viper-lang plug-in provides:
  - basic support for editing Viper files (automatic filetype detection,
    identation, syntax highlighting)
  - snippets for faster editing

==============================================================================
SNIPPETS                              *viper-snippets* *vim-viper-lang-snippets*

This plugin provides the following snippets:
  - {acc}: access modifier
  - {imp}: import declaration
  - {def}: macro definition
  - {pred}: predicate definition
  - {dom}: domain definition
  - {ax}: axiom definition
  - {field}: field declaration
  - {fun}: function definition
  - {met}: method definition
  - {if}: if statement
  - {else}: else conditional branch
  - {while}: while loop
  - {forall}: for all quantifier
